Project Name: FoundIt

PROJECT DESCRIPTION
FoundIt is a full-stack, capstone-level web application designed to help communities report and locate lost items. Built using a secure, decoupled architecture, it pairs a fast, responsive vanilla JavaScript frontend with a robust Node.js and Express.js RESTful API.

The application utilizes a relational PostgreSQL database for structured data management, relies on cryptographically secure password hashing (bcrypt) with JWT session tokens for authentication, and features direct, role-based administrative moderation overrides.
